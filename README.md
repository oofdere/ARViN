# ARViN
(A)ugmented (R)eality (Vi)sual (N)ovel is a bodged-together collection of code and assets used in the production of VN_irl, in order to save me from staring at After Effects for hours on end.

## Requirements
 - Computer or PDA
 - Modern Web Browser
 - Modern Operating System (only Windows is tested and most things will likely only work on Windows w/o porting work; not a technical limitation, just because I'm lazy and tstrapped for time; this should change soonâ„¢)
 - Webcam ([or something that fakes one](https://www.elgato.com/en/gaming/cam-link-4k))
 - Letter Paper
 - Printer
 - Printer Ink/Toner
